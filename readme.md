# Nodejs CLI TOOLS
## watch & copy command:
	using [commander](https://github.com/tj/commander), [fs-extra](https://github.com/jprichardson/node-fs-extra) and [chokidar](https://github.com/paulmillr/chokidar) to create a simple file and directory monitor & synchronize tool.
	mycli watch-cp <source> [destinations]
